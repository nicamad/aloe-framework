# aloe-framework

A framework for defining, evaluating, and operating purpose-built agents through `aloe.yaml`.

## Core ideas

- `aloe.yaml` defines an agent's purpose, inputs, actions, integrations, evaluation, and handoff.
- Starling Operator and Aloe Operator are reference agents built on top of this framework.
- Solarly Operator is the whiteglove orchestration layer that uses these agents in delivery workflows.

## Initial scope

- execution envelope
- execution mode resolution
- action registry
- action state resolver
- connector lifecycle model
- `aloe.yaml` schema
